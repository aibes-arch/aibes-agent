# AI 编程与智能体（Vibe Coding & Agentic Programming）

> 从"写代码"到"描述需求让 AI 写代码"，编程范式正在发生根本性转变。

## 核心概念

| 概念 | 定义 | 代表工具 |
|------|------|---------|
| **Vibe Coding** | 用自然语言描述意图，AI 生成代码，人负责审阅和调整 | Cursor, Windsurf, GitHub Copilot |
| **Agentic Programming** | AI 代理自主执行多步骤任务，调用工具，迭代完成目标 | Claude Code, OpenAI Codex, Devin |
| **Augmented LLM** | 增强型大语言模型，具备检索、工具调用、记忆能力 | GPT-4 + MCP, Claude + Tools |
